## Getting Started

### Prerequisites
- Node.js v18+
- Git
- Supabase account
- Stripe account

### Installation
```bash
git clone https://github.com/your-username/TextCentre.git
cd TextCentre
npm install
```

### Configuration
1. Copy `.env.example` to `.env`
2. Fill in your Supabase and Stripe keys

### Running Locally
```bash
npm run dev
```
